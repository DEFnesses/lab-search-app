# 🧪 Лабораторный поиск

Веб-приложение для поиска лабораторных исследований пациента по номеру СНИЛС.  
Данные хранятся в локальном JSON-файле (mock-данные). Проект написан на React + TypeScript с использованием Vite.

---

## 🛠️ Технологии

- **React 19** — библиотека для построения интерфейсов
- **TypeScript** — типизация кода
- **Vite** — сборка проекта
- **CSS (чистый)** — стилизация без сторонних библиотек
- **Адаптивная вёрстка** — поддержка мобильных устройств

---

## 📦 Установка и запуск

```bash
# Клонируем репозиторий
git clone https://github.com/DEFnesses/lab-search-app.git

# Переходим в папку проекта
cd lab-search-app

# Устанавливаем зависимости
npm install

# Запускаем проект в режиме разработки
npm run dev

Функциональность

  🔍 Поиск пациента по СНИЛС — номер можно вводить в любом формате (с дефисами, пробелами или без)
  📋 Таблица с анализами — отображение показателей, результатов и норм
  🧮 Количество найденных анализов — автоматический подсчёт
  ⏳ Состояние загрузки — индикатор во время поиска
  ❌ Обработка ошибок — сообщения выводятся прямо на страницу
  ⌨️ Обработка Enter — поиск по нажатию клавиши
  📱 Адаптивная вёрстка — корректное отображение на всех устройствах
  📂 Моковые данные — 7 пациентов с разными анализами (включая пустые)
